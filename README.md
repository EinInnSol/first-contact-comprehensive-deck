## Quick Start

This is a static HTML presentation - no build required!

### Local Development

```bash
python3 -m http.server 3000
```

Open [http://localhost:3000](http://localhost:3000)

### Deploy to Vercel

1. Push to GitHub
2. Go to vercel.com
3. Import this repository
4. Deploy (no build configuration needed!)
